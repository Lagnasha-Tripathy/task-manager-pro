# Task Manager Pro

A modern task management application built with React.js that enables users to create, manage, and track daily tasks efficiently through a clean and responsive interface.

## Features

* Create new tasks
* Mark tasks as completed
* Undo completed tasks
* Delete tasks
* Persistent task storage using Local Storage
* Real-time task statistics

  * Total Tasks
  * Completed Tasks
  * Pending Tasks
* Responsive user interface

## Technology Stack

## Installation

### Clone the Repository

```bash
git clone https://github.com/Lagnasha-Tripathy/task-manager-pro.git
```

### Navigate to the Project Directory

```bash
cd task-manager-pro/frontend
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

## Application Overview

The application uses React state management to handle task operations and Local Storage to persist data between browser sessions. Users can add, complete, undo, and delete tasks while viewing real-time task statistics.
